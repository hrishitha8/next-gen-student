# Next-Gen Learning Dashboard

A modern student dashboard built using:

- Next.js 16
- Supabase
- Tailwind CSS
- Framer Motion
- Lucide React

Features:
- Bento Grid Dashboard
- Sidebar Navigation
- Live Supabase Data
- Animated Course Cards
- Activity Tile
- Dark Theme UI
- Responsive Layout

Setup:

1. Install dependencies

npm install

2. Add environment variables

NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=

3. Run

npm run dev
